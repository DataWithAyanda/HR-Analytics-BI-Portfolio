# Stage 1: Data Foundations (0–9 Months)

**Identity:** Junior HR Data Analyst  
**Goal:** Automate HR tasks, master Python & SQL, maintain portfolio visibility in GitHub.

---

## 📌 Overview

Stage 1 introduces foundational **Python, Pandas, CSV handling, and basic visualization**, gradually building to a full **Attrition Analysis capstone**.  
Each week contains **mini-projects, exercises, and a weekly portfolio-ready build**.

**Git Practices:**  
- Minimum 3 commits per week  
- Merge when stable  
- Weekly README updates with screenshots, outputs, and notes  

---

## 🛠 Skills Demonstrated

- **Python Fundamentals:** Variables, data types, loops, functions  
- **Data Structures:** Lists, dictionaries, iterations  
- **File Handling:** CSV reading/writing, data cleaning  
- **Pandas:** DataFrames, filtering, grouping, merging  
- **Visualization:** Seaborn / Matplotlib basics  
- **Mini-projects & Capstones:** End-to-end HR analytics scripts
