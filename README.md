## 📌 Overview

This repository tracks my **24-month HR Data Analytics roadmap**, from foundational Python and SQL to cloud-based automation, MLOps, and autonomous workforce analytics pipelines.

**Portfolio Highlights:**  
- **Stage 1:** Python & Pandas Foundations → Payroll Calculator, Bonus Eligibility Engine, Employee Roster Analyzer, Attrition Analysis  
- **Stage 2:** SQL & Automation → Data warehouse, API integration, Retention Oracle  
- **Stage 3:** Specialization → Containerized pipelines, advanced DAGs, bias auditing, autonomous HR analytics engine  

> Each folder contains **weekly builds, mini-projects, and capstones**. Screenshots and outputs are stored in `Portfolio_Assets` for portfolio-ready visibility.
